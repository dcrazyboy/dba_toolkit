# 🤔 L'idée de base

Monté un github pour parle de base de données ou d'outils, implique de choisr un OS sur le PC. Là ou cela se complique c'est le chooix de l'OS de base et ce que cela implique dans la gestion. Ce github  présente des olution plus ou moin dépendantes du système, dans les outils ou l'installation de la base et son organisation.
J'ai choisit un PC Portable qui me sert de base pour tous ces documents. Comme toujours c'est une proposition, vous pouvez choisir celui que vous voulez, mais voici sa cofiguration pour mes périgrinations.

Donc mon modele de base : 

- PC portable : portable Asus Rog
- Carte Graphisue : NVIDIA GeForce RTX 3070 (8 Go VRAM)
- Processeur : AMD RYZEN 7 (6000 series)
- Memoire : 32G
- DD : 1 To
- webcaam : generique  usb
- DD externe : 4 To
- Clef USB : 128 Gb
